Formulaires

Les formulaires permettent à vos visiteurs de saisir des informations, appuyer sur des boutons, ... etc.

Cependant le traitement des données saisies devra se faire dans un autre langage.


En attendant, l'affichage de ces éléments de formulaire se fait bel et bien en HTML !

Insérer un formulaire

L'ajout d'un formulaire se fait avec la balise   <form>.


<h2>Veuillez vous identifier :</h2>

<form><!-- Début du formulaire -->
    
    <!-- Contenu de mon formulaire ici -->

</form><!-- Fin du formulaire -->

À cette balise   <form>   s'ajoute 2 attributs :

  1. method, qui aura pour valeur :
    • get
    • post
  2. action, qui aura pour valeur :
    • l'URL cible qui recevra les données

Exemple :


<form method="post" action="cible.php">
    ...
</form>

Elements simples

Zones de texte


<input type="text" name="prenom">
Résultat :


L'attribut   name   permet d'indiquer le nom du champs pour le traitement côté serveur.

Bonnes pratiques : le label

Le label permet d'indiquer un texte donnant une information sur le champs.


<form method="post" action="cible.php">
    
    <label for="prenom">Indiquez votre prénom :</label>
    <input type="text" name="prenom" id="prenom">

</form>
Résultat :


Attributs supplémentaires

  • value
  • size
  • maxlength
  • placeholder

<form method="post" action="cible.php">
    
  <label for="nom">Indiquez votre nom :</label>
  <input type="text" name="nom" id="nom"
         maxlength="15" size="50" placeholder="Exemple : DUPONT">

</form>
Résultat :

Les éléments d'options

On peut également trouver d'autres types d'éléments que des zones de texte :

  • cases à cocher :
  • boutons radios :

Les cases à cocher


<form method="post" action="cible.php">
    
  <input type="checkbox" name="newsletter" id="newsletter">
  <label for="newsletter">Recevoir la newsletter ?</label>
  <br>

  <input type="checkbox" name="magazine" id="magazine" checked>
  <label for="magazine">S'abonner à "tartanpion magazine" ?</label>

</form>
Résultat :

Les boutons radios

Elles permettent de faire un seul choix parmi une liste de possibilités.
Elles doivent être organisées en groupes.


<form method="post" action="cible.php">
    
 <p>Soit 0, 1, 1, 2, 3, 5, 8, 13, 21, N. Que vaut 'n' ?</p>

 <input type="radio" name="suite" id="r1" value="28">
 <label for="r1">28</label>

 <input type="radio" name="suite" id="r2" value="34">
 <label for="r2">34</label>

 <input type="radio" name="suite" id="r3" value="36">
 <label for="r3">36</label>

</form>
Résultat :

Soit 0, 1, 1, 2, 3, 5, 8, 13, 21, N. Que vaut 'n' ?

Les listes déroulantes (1/2)


<form method="post" action="cible.php">

    <option value="FRA">France</option>
    <option value="ITA">Italie</option>
    <option value="ESP">Espagne</option>
    <option value="ALL">Allemagne</option>
  </select>

</form>

Les listes déroulantes (2/2)

On peut également créer des groupes au sein de la liste


<form method="post" action="cible.php">
    
  <label for="dest">Choisissez votre destination :</label>
  <select name="dest" id="dest">
    <optgroup label="Europe">
      <option value="FRA">France</option>
      <option value="ITA">Italie</option>
      <option value="ESP">Espagne</option>
      <option value="ALL">Allemagne</option>
    </optgroup>
    <optgroup label="USA">
      <option value="FLO">Floride</option>
      <option value="CAL">Californie</option>
    </optgroup>
  </select>

</form>

Elements de validation

Il existe des éléments de type "bouton" pour donner un côté intéractif à un formulaire.

Boutons de validation


<form method="post" action="cible.php">
  <label for="nom">Entrez votre nom</label>
  <input type="text" name="nom" id="nom">

  <label for="prenom">Entrez votre prénom</label>
  <input type="text" name="prenom" id="prenom">

  <input type="submit" name="valid" value="Envoyer le formulaire">
  <input type="reset" name="reinit" value="Réinitialiser le formulaire">
</form>

Autres boutons

Il est aussi tout à fait possible d'utiliser des boutons qui n'ont aucune action par défaut.

Ces actions seront ensuite établies à l'aide du langage JavaScript


<input type="button" id="mon-btn" name="mon-btn" value="Mon bouton">

<button id="btn-customisable" name="btn-customisable">
  <!-- On peut mettre ici n'importe quoi -->
  <h4>Cliquez !</h4>
  <img src="btn.png" alt="Sign In">
</button>
