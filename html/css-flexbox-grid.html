Flexbox module

Nouveau contexte d'affichage et d'alignement de blocs

float: left | right

display: block | inline | inline-block | table

position: relative | absolute | fixed

display: flex

Flexbox est un nouveau mode de positionnement, introduit via la propriété display, permettant de créer un contexte général d'affichage sur un parent et d'en faire hériter ses enfants :

Avantages

  • Distribution en ligne ou en colonne
  • Alignements horizontaux et verticaux
  • Gestion des espacements
    (beat that, table!)
  • Ordre d'affichage des éléments
    (indépendamment de leur position dans le DOM)
