Transitions

Utiliser les transitions CSS

Une transition en CSS permet d'interpoler les valeurs de propriétés d'un élément depuis un état A vers un état B

Exemple de transition sur background-color lors d'un :hover

Hover me!

On utilise la propriété CSS transition,
dont voici la syntaxe :


						transition: <propriété> <durée> <timing> <délai>
					

						transition: <propriété> <durée> <timing> <délai>
  • <propriété> : Nom de la propriété à animer
  • <durée> : Durée de l'animation (s ou ms)
  • <timing>* : Fonction de timing
  • <délai>* : Délai avant déclanchement (s ou ms)

* (optionnel)

Exemple d'utilisation sur un lien :


						a {
							…
							background-color: rgba(0, 100, 200, .8);
							transition: background-color 1s;
						}

						a:hover {
							background-color: rgba(0, 200, 100, .8);
						}
Mon lien

On peut utiliser all pour animer toutes les propriétés changeantes


						a {
							…
							background-color: rgba(0, 100, 200, .8);
							color: white;
							transition: all 1s;
						}

						a:hover {
							color: black;
							background-color: rgba(0, 200, 100, .8);
						}

Animations

http://www.w3.org/TR/css3-animations/

Les animations permettent de faire des choses plus poussées que les transitions.

Par exemple, on peut faire boucler une animation à l'infini.


Pour définir une animation, il faut d'abord
définir son comportement !

Le comportement régit la façon dont va réagir l'objet animé depuis un état 0% vers un état 100%

Un comportement se définit à l'aide du mot-clé @keyframes

(À préfixer par @-webkit-keyframes pour Chrome)


						@keyframes rebond
						{
							0%   { margin-left: 0; }
							50%  { margin-left: 900px; }
							100% { margin-left: 0; }
						}

Une fois le comportement défini, on peut l'utiliser pour animer n'importe quel élément HTML.

La propriété CSS pour utiliser des animations est :
animation


(À préfixer pour Chrome ! http://shouldiprefix.com/#animation)
-webkit-animation
animation

Syntaxe

animation: nom durée fonctionTiming délai répétitions direction

https://developer.mozilla.org/fr/docs/CSS/animation

Exemple d'utilisation sur une <div> carrée :


						.carre {
							width: 100px;
							height: 100px;
							background-color: rgba(0, 100, 255, .7);
							
							-webkit-animation: rebond 3s linear infinite;
							animation: rebond 3s linear infinite;
						}
