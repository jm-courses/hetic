Flexbox Module

Contexte d'affichage et d'alignement de blocs

Proposé en 2009 et recommandé à partir de 2012

float: left | right

display: block | inline | inline-block | table

position: relative | absolute | fixed | sticky

display: flex

Flexbox est un nouveau mode de positionnement, introduit via la propriété display, permettant de créer un contexte général d'affichage sur un parent et d'en faire hériter ses enfants :

Avantages

  • Distribution en ligne ou en colonne
  • Alignements horizontaux et verticaux
  • Gestion des espacements
  • Ordre d'affichage des éléments
    (indépendamment de leur position dans le DOM)

On définit la règle display: flex; sur l'élément parent

Les éléments enfants se positionneront alors comme des flex items


								

									
flex item 1

									
flex item 2

									
flex item 3

								

							

								.parent {
									display: flex;
								}

Un élément flex peut positionner ses enfants directs sur 2 axes :

  • l'axe principal
  • l'axe secondaire

Par défaut, l'axe principal est horizontal, et l'axe secondaire est vertical

Mais cela peut être changé !

Mais cela peut être changé !


						.parent {
							flex-direction: column;
						}

L'axe principal est maintenant l'axe vertical, et le secondaire est l'axe horizontal

justify-content

Cette propriété définit la disposition des éléments de l'axe principal


							.parent {
								display: flex;
								justify-content: space-between;
							}

align-items

Cette propriété définit la disposition des éléments de l'axe secondaire


							.parent {
								display: flex;
								justify-content: flex-start;
								align-items: center;
							}

flex: <n>

Se place sur les enfants (flex items)


						.parent > * {
							flex: 1;
						}

order: <n>

Définit l'ordre d'affichage
